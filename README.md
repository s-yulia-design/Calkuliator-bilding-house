# Калькулятор сметы строительства ИЖД

Веб-приложение для расчёта стоимости строительства индивидуального дома: этапы, материалы, работы, итоги. Есть простой API (SQLite), админка прайса и шаблон с объёмами из проекта «Нигметова Мадина».

## Запуск

Требования: **Node.js 22+** (используется встроенный SQLite).

```bash
npm install
npm run dev
```

Откроется:

- Калькулятор: http://localhost:5173
- С телефона в той же Wi‑Fi: `http://IP-вашего-ПК:5173`
- API: http://localhost:3001
- Админка: http://localhost:5173/admin

Пароль админа по умолчанию: `admin123` (файл `.env`).

## Возможности

- Создание пустой сметы или из шаблона ИЖД
- Этапы и позиции: количество × цена материала + количество × расценка работы
- Каталог материалов с ценами (редактирование в админке)
- Ссылка на проект для открытия на телефоне
- Экспорт CSV и печать
- PWA (можно установить на домашний экран)

## Структура

```
client/   — React (Vite) UI + PWA
server/   — Hono API + SQLite
shared/   — общие типы и формулы
data/     — файл БД (создаётся автоматически)
```

## Продакшен

```bash
npm run build
npm start
```

Сервер раздаёт собранный `client/dist` и API на порту `PORT` (по умолчанию 3001).

Смените в `.env`: `ADMIN_PASSWORD`, `JWT_SECRET`, `CLIENT_ORIGIN`.

На главной: блок «Новый проект» и список «Уже рассчитанные» — без id и ключей доступа.
