# Калькулятор сметы строительства ИЖД

Веб-приложение для расчёта стоимости строительства индивидуального дома: этапы, материалы, работы, итоги. Работает **в браузере** (данные в localStorage), без сервера — можно открыть на GitHub Pages.

## Онлайн

После публикации:  
**https://s-yulia-design.github.io/Calkuliator-bilding-house/**

## Локальный запуск

Требования: Node.js 22+.

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

**Настройка цен:** http://localhost:5173/prices  
(без пароля; можно указать регион и дату актуальности цен)

## Возможности

- Новый проект или шаблон ИЖД 001
- Этапы, позиции, доп. материалы/работы и заметки
- Раздел «Инженерные системы и отделка»
- Каталог цен и правка шаблона в «Настройке цен»
- Экспорт CSV и печать
- PWA (установка на телефон)

## Структура

```
client/   — React (Vite) + PWA, всё хранится в браузере
shared/   — типы, формулы, шаблон
server/   — опциональный API (для разработки, не нужен для GitHub Pages)
```

## Деплой на GitHub Pages

При push в `main` срабатывает Actions → публикуется приложение (не README).

В настройках репозитория (**Settings → Pages**) выберите одно из:

1. **Source: GitHub Actions** (предпочтительно), или  
2. **Deploy from a branch** → ветка `gh-pages` → папка `/`

Одна и та же ссылка работает на компьютере и на телефоне:

https://s-yulia-design.github.io/Calkuliator-bilding-house/

Сборка с префиксом `/Calkuliator-bilding-house/`.
